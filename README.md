## ⌨ Como executar o projeto

```bash
# Clonar o repositório
git clone https://github.com/giow/proffy-nlw

# Entrar no projeto
cd proffy-nlw

# Baixar as dependências
npm i

# Ligar a aplicação
npm run dev
```

Abra o seu navegador e acesse `http://localhost:5500/`
